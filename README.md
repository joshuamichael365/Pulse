# Pulse

An AI-powered habit tracking and wellness journal for iOS.

## What it does
Pulse unifies habit tracking and daily wellness check-ins into a personal corpus, 
then uses a RAG (Retrieval-Augmented Generation) pipeline to surface personalised 
insights grounded in the user's own history — not generic AI advice.

## Tech Stack
- SwiftUI + SwiftData
- Supabase + pgvector
- HealthKit
- Apple Foundation Models
- WidgetKit
- Xcode Cloud
